# student-productivity-tracker
AI Study Assistant

## Overview
The AI Study Assistant is a productivity-focused tool designed to help students plan and optimize their study schedules. It uses basic data-driven logic to prioritize tasks and improve time management.

## Motivation
As a high school student, I observed that many students struggle with managing multiple subjects efficiently. This project was built to explore how computational thinking and simple AI concepts can be applied to real-world academic challenges.

## Features
- Task prioritization based on deadlines and difficulty
- Study schedule generation
- Simple prediction of workload distribution
- User-friendly command-line interface

## How It Works
